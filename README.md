# practice_php_bbs
 
ひと言掲示板
 
## 簡単な説明
 
PHP学習の一環で作成したひと言掲示板です。

サンプルコードを見ながら作成しました。

ユーザはユーザ名とメッセージをセットで書き込むことができます。

管理者はメッセージの編集および削除、メッセージの一括ダウンロードができます。

## 機能
 
- 機能1　メッセージの投稿
- 機能2　メッセージの削除　※管理者のみ
- 機能3　メッセージの編集　※管理者のみ
- 機能4　メッセージの一括ダウンロード　※管理者のみ

## 開発環境

- MAMP 5.4
- macOS Mojave 10.14.4
- Apache 2.2.34 
- MySQL 5.7.26
- PHP 7.3.6
